Everything your SOC needs

Full-stack incident response,
out of the box.

Every module ships ready to use. No professional services, no 6-month implementation, no hidden costs.

Case Management

Full incident lifecycle from alert to closure. Entity tracking, chronological timeline, evidence locker, severity classification, and role-based access for Admin, Analyst, Approver, and Viewer roles.

AI Analyst (Claude API)

Automatic MITRE ATT&CK mapping, hunt queries in SPL/KQL/EQL, severity rationale, playbook recommendation, and a conversational chatbot — all from the case context. Prompt caching keeps costs low.

Governed Playbook Engine

Dry-run preview → approval gates → live execution → rollback → verification. Blast-radius limits cap what a playbook can touch. Safety constraints enforced at the engine level — impossible to skip.

Daily Threat Intelligence

Automated pipeline pulls CISA KEV, URLhaus, and AlienVault OTX every morning at 06:00 UTC. Claude synthesizes them into plain-English briefings and proposes new detections and playbook updates.

Weekly Security Patches

Risk-tiered patch bundles auto-generated every Sunday. Low-risk changes auto-apply; medium and high require analyst or approver sign-off. Full rollback manifest generated before any change lands.

NIST CSF 2.0 Compliance

Every case is mapped across all six NIST CSF 2.0 functions: Govern, Identify, Protect, Detect, Respond, Recover. Org-wide gap report with specific, actionable improvement recommendations.

MTTR & Metrics Dashboard

Average, p50, and p95 MTTR by severity. 8-week trend chart. SLA benchmark comparison. Stale cases breaching the 24-hour threshold. Management-ready reporting without exporting to a spreadsheet.

IOC Auto-Enrichment

VirusTotal and AbuseIPDB lookups run in the background the moment an entity is created — never blocks case ingestion. Color-coded threat badges, geo info, and ASN data displayed inline on every entity.

Immutable Audit Log

Every action by every user is logged forever, append-only by design. Exportable audit packet per incident. SOC 2 and compliance-auditor ready out of the box — no configuration required.

Simple by design

From alert to closed case
in minutes.

ThreatCommand guides your team through every step of the NIST incident response lifecycle — automatically, with AI support at every decision point.

1
Alert arrives
Sentinel, Splunk, CrowdStrike, or Elastic webhook hits the ingest API. Auto-deduplicated and converted to a managed case.
2
AI analyzes
Claude maps TTPs to MITRE ATT&CK, explains severity, and recommends the right playbook — before an analyst even opens the case.
3
Playbook executes
Dry-run preview shows exactly what will happen. Approval gates require sign-off on high-impact steps. Blast-radius limits prevent accidental scope creep.
4
Case closes with audit trail
Incident report, NIST CSF audit packet, and MTTR metrics update automatically. Auditors get the full picture with one click.
# Clone and run in under 5 minutes

git clone https://github.com/reesiant737-creator/threatcommand.git
cd threatcommand
cp .env.example .env
# Add your ANTHROPIC_API_KEY to .env
pip install -r requirements.txt
python run.py

✓ ThreatCommand running at http://localhost:5001

# Load realistic demo data
flask seed-demo

✓ 5 cases seeded (BEC, ransomware, OAuth abuse, ...)
✓ 2 playbook runs (1 completed, 1 pending approval)
✓ 3 weeks of threat intel history
✓ Full audit trail ready

# Or one-click Docker deploy
docker-compose up -d

✓ Production-ready. Persistent SQLite (or swap DATABASE_URL for Postgres)

Gold-standard response

5 production playbooks. Day one.

Every playbook was written to real DFIR standards — not generic checklists. Dry-run preview, approval gates, blast-radius caps, and full rollback built in.

Pack A · BEC
Suspicious Inbox Forwarding

Detects and remediates mailbox rules silently forwarding mail to external addresses — the #1 BEC indicator. Sessions revoked, rule disabled, evidence preserved.

MITRE T1564.008
Pack A · BEC
OAuth Consent Abuse

Revokes malicious OAuth app grants, inventories all consented apps, and blocks future consent for unapproved applications. Covers pass-the-token and consent phishing.

MITRE T1550.001
Pack B · Ransomware
Ransomware Execution

Network isolation, memory capture, shadow copy preservation, backup integrity check, and credential rotation — in the right order, with the right approvals at every step.

MITRE T1486 · T1490
Pack B · Ransomware
Lateral Movement

Cuts off credential hopping via SMB/RDP, forces re-authentication on pivot hosts, and revokes service account tokens across affected segments without taking down production.

MITRE T1021
Pack C · Insider
Suspicious Admin Activity

Handles valid-account abuse: off-hours logins, privilege escalation, data staging. Preserves evidence before any access revocation so the audit trail stays intact.

MITRE T1078
Build your own

Playbooks are plain JSON. Write a step, add an approval gate, set a blast radius, and deploy. No proprietary scripting language to learn.
